# Wrapper API 

Some generic cody extracted from  by a number of api wrapper gems. Internal used only.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'wrapi'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install wrapi
